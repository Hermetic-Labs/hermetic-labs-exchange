# Hero Image Placeholder

Replace this file with an actual PNG image for your package.

## Recommended Specifications

- **Size:** 1600 x 900 pixels (16:9 aspect ratio)
- **Format:** PNG or WebP
- **File size:** Under 500KB (optimize for web)

## What to Include

Your hero image should:

- Show your package's main interface
- Use dark backgrounds (matches EVE OS theme)
- Include key features or UI elements
- Be visually striking for the marketplace

## Design Tips

1. Take a screenshot of your package in action
2. Add subtle branding or text overlay
3. Use gradients or glows for visual interest
4. Ensure text is readable at small sizes

## Naming

Keep the filename as `hero.png` to match the manifest.json media reference.
